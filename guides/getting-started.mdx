## 회원가입

GrowFit을 시작하려면 먼저 계정을 만들어야 합니다. 아래 단계를 따라 가입을 진행하세요.

<Steps>
  <Step title="웹사이트 접속">
    브라우저를 열고 **[growfit.kr](https://growfit.kr)** 에 접속합니다.
  </Step>
  <Step title="이메일 인증">
    회원가입 페이지에서 이메일 주소를 입력하고 **인증 코드 발송** 버튼을 클릭합니다. 입력한 이메일로 발송된 인증 코드를 입력하여 이메일을 확인합니다.

    <Warning>
      인증 코드는 발송 후 **10분 이내에** 입력해야 합니다. 시간이 초과되면 재발송 버튼을 눌러 새 코드를 받으세요.
    </Warning>
  </Step>
  <Step title="역할 선택 — 수강생 선택">
    이메일 인증 후 역할을 선택하는 화면이 나타납니다. **수강생**을 선택하세요.

    <Info>
      역할에 따라 가입 방식이 다릅니다. 아래 역할 구분 안내를 참고하세요.
    </Info>
  </Step>
  <Step title="강의 등록 — 초대 코드 입력">
    강사로부터 받은 **초대 코드**를 입력하여 강의에 등록합니다. 초대 코드는 담당 강사에게 문의하세요.

    <Tip>
      초대 코드를 아직 받지 못했다면, 가입 후 나중에 **프로필 > 수강 강의** 메뉴에서도 코드를 입력할 수 있습니다.
    </Tip>
  </Step>
</Steps>

## 역할 구분

GrowFit에는 두 가지 역할이 있습니다. 가입 시 선택하는 역할에 따라 사용 가능한 기능이 달라집니다.

<CardGroup cols={2}>
  <Card title="훈련생 (수강생)" icon="user-graduate">
    **가입 즉시 사용 가능**

    - LLM 실습, 프롬프트, RAG, MCP, 에이전트 빌더 등 모든 학습 기능 이용
    - 강의에 등록하여 수업 참여
    - 학습 기록 및 성과 대시보드 확인
    - 별도의 승인 절차 없이 바로 사용 가능
  </Card>
  <Card title="강사" icon="chalkboard-user">
    **관리자 승인 필수**

    - 강의 개설 및 관리
    - 수강생 초대 및 모니터링
    - 학습 현황 통계 열람
    - 교육 콘텐츠 커스터마이징
    - **가입 후 관리자의 승인이 완료되어야 강사 기능 활성화**
  </Card>
</CardGroup>

<Warning>
  강사 계정은 관리자 승인 전까지 강사 전용 기능(강의 관리, 수강생 모니터링 등)을 사용할 수 없습니다. 승인 완료 전에는 수강생과 동일한 기능만 이용 가능합니다.
</Warning>

## 로그인 및 계정 보안

<img
  src="/images/manual-slide10-35.png"
  alt="로그인 화면"
  className="rounded-xl w-full my-4"
/>

가입 완료 후 **growfit.kr**에 접속하면 로그인 화면이 나타납니다. 가입 시 사용한 이메일과 비밀번호를 입력하여 로그인하세요.

<AccordionGroup>
  <Accordion title="비밀번호 분실 시" icon="key">
    로그인 화면에서 **비밀번호 찾기**를 클릭하고 가입한 이메일을 입력하면 재설정 링크가 발송됩니다.
  </Accordion>
  <Accordion title="로그아웃 방법" icon="right-from-bracket">
    화면 좌측 하단의 **프로필 아이콘**을 클릭하면 메뉴가 나타납니다. **로그아웃** 버튼을 클릭하여 안전하게 계정에서 나올 수 있습니다.

    공용 컴퓨터에서 사용한 경우 반드시 로그아웃하세요.
  </Accordion>
  <Accordion title="비밀번호 변경" icon="lock">
    **프로필 > 계정 설정 > 비밀번호 변경** 메뉴에서 현재 비밀번호를 확인 후 새 비밀번호로 변경할 수 있습니다.
  </Accordion>
</AccordionGroup>

## 사이드바 메뉴 상세

사이드바에는 GrowFit의 7가지 핵심 메뉴가 있습니다. 각 메뉴는 학습 단계에 따라 순서대로 배치되어 있습니다.

<AccordionGroup>
  <Accordion title="① 내 프로젝트 (Step 1)" icon="folder-open">
    GrowFit에서 모든 작업은 **프로젝트** 단위로 관리됩니다.

    - 새 프로젝트 생성
    - 기존 프로젝트 열기 및 편집
    - 프로젝트별 AI 대화 기록 관리
    - 프로젝트 공유 및 복제

    프로젝트는 학습 주제 또는 업무 목적별로 구분하여 관리하는 것을 권장합니다.
  </Accordion>
  <Accordion title="② LLM 실습 (Step 2)" icon="comments">
    <img
      src="/images/manual-slide13-30.png"
      alt="LLM 실습 메뉴"
      className="rounded-xl w-full my-2"
    />

    다양한 AI 모델에 직접 질문하고 결과를 비교하는 핵심 실습 공간입니다.

    - 19종 AI 모델 중 선택하여 대화
    - Temperature, Top-P 파라미터 실시간 조정
    - 모델별 응답 비교
    - 대화 기록 저장 및 내보내기
  </Accordion>
  <Accordion title="③ 프롬프트 (Step 3)" icon="pen-nib">
    AI에게 역할과 행동 방식을 지시하는 프롬프트를 설계하고 관리합니다.

    - 시스템 프롬프트 작성 및 저장
    - Few-shot 예시 추가
    - 프롬프트 버전 관리
    - 에이전트 빌더에서 재사용 가능한 템플릿 저장
  </Accordion>
  <Accordion title="④ 지식베이스 RAG (Step 4)" icon="database">
    <img
      src="/images/manual-slide15-1.png"
      alt="지식베이스 RAG 메뉴"
      className="rounded-xl w-full my-2"
    />

    문서를 업로드하여 AI가 해당 내용을 참조하도록 하는 지식베이스를 구성합니다.

    - PDF, Word, 텍스트 파일 업로드
    - 지식베이스 생성 및 관리
    - 업로드된 문서 미리보기
    - 검색 결과 품질 테스트
  </Accordion>
  <Accordion title="⑤ MCP (Step 5)" icon="plug">
    <img
      src="/images/manual-slide16-0.png"
      alt="MCP 메뉴"
      className="rounded-xl w-full my-2"
    />

    외부 도구를 AI에 연결하여 AI의 활용 범위를 확장합니다.

    - MCP 서버 목록 확인
    - 외부 도구 연결 설정
    - 연결 상태 테스트
    - 에이전트 빌더와 연동
  </Accordion>
  <Accordion title="⑥ 에이전트 빌더 (Step 6)" icon="diagram-project">
    프롬프트, 지식베이스, MCP를 조합한 복잡한 AI 워크플로우를 시각적으로 설계합니다.

    - 드래그 앤 드롭 노드 배치
    - 노드 간 연결선 설정
    - 워크플로우 실행 및 테스트
    - 에이전트 저장 및 공유
  </Accordion>
  <Accordion title="⑦ 학습 기록" icon="chart-line">
    나의 학습 활동과 성과를 확인하는 대시보드입니다.

    - 전체 학습 시간 통계
    - 단계별 학습 완료 현황
    - AI 모델별 사용 빈도
    - 기간별 학습 활동 그래프
  </Accordion>
</AccordionGroup>

## 프로필 및 설정

<img
  src="/images/manual-slide19-17.png"
  alt="프로필 설정 화면"
  className="rounded-xl w-full my-4"
/>

사이드바 하단의 **프로필 아이콘**을 클릭하면 계정 설정 메뉴에 접근할 수 있습니다.

<Tabs>
  <Tab title="계정 정보">
    이름, 이메일 등 기본 프로필 정보를 확인하고 수정할 수 있습니다.

    - 표시 이름 변경
    - 프로필 사진 업로드
    - 이메일 주소 확인
  </Tab>
  <Tab title="비밀번호 변경">
    현재 비밀번호를 입력하고 새 비밀번호를 설정합니다.

    <Warning>
      보안을 위해 비밀번호는 **8자 이상**, 영문·숫자·특수문자를 조합하여 설정하세요.
    </Warning>
  </Tab>
  <Tab title="수강 강의">
    현재 등록된 강의 목록을 확인하고, 새로운 강의에 등록하기 위한 초대 코드를 입력할 수 있습니다.

    - 현재 수강 중인 강의 목록
    - 강의 진도율 확인
    - 초대 코드로 새 강의 등록
  </Tab>
  <Tab title="테마 설정">
    GrowFit 인터페이스의 테마를 변경합니다.

    - **라이트 모드**: 밝은 배경의 기본 테마
    - **다크 모드**: 어두운 배경의 테마 (눈의 피로 감소)
    - **시스템 설정 따르기**: 운영체제 설정에 맞춰 자동 전환
  </Tab>
</Tabs>

## 시작하기 체크리스트

GrowFit을 처음 시작하기 위해 아래 항목을 순서대로 완료하세요.

<Steps>
  <Step title="회원가입 완료">
    [growfit.kr](https://growfit.kr) 에서 이메일 인증 후 계정 생성
  </Step>
  <Step title="초대 코드 입력">
    담당 강사로부터 받은 초대 코드를 입력하여 강의 등록
  </Step>
  <Step title="화면 구성 파악">
    사이드바, 메인 영역, 상단 메뉴의 역할 확인
  </Step>
  <Step title="첫 번째 프로젝트 생성">
    **내 프로젝트(Step 1)** 메뉴에서 새 프로젝트 만들기
  </Step>
  <Step title="LLM 실습 시작">
    **LLM 실습(Step 2)** 메뉴에서 AI 모델에게 첫 질문 던져보기
  </Step>
  <Step title="학습 경로 확인">
    사이드바의 나머지 메뉴(프롬프트 → RAG → MCP → 에이전트 빌더)를 순서대로 탐색
  </Step>
</Steps>

<Tip>
  막히는 부분이 있으면 **[FAQ](/guides/faq)** 페이지를 확인하거나 고객 지원 이메일로 문의하세요.
</Tip>
