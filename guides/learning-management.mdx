## 학습 성과 대시보드

GrowFit에서 어떤 기능을 얼마나, 어떻게 활용했는지 한눈에 파악할 수 있습니다. 대시보드는 5개 영역으로 구성됩니다.

<Tabs>
  <Tab title="AI 실습 학습">
    **LLM 활용 현황을 상세히 추적합니다.**

    - 모델별 사용량
    - 대화 모드 (단일 / 비교)
    - 파라미터 설정 이력
    - Few-shot 활용 현황
    - 파일 첨부 횟수
  </Tab>
  <Tab title="지식베이스 학습">
    **RAG 활용 현황을 추적합니다.**

    - 문서 업로드 수
    - 생성된 총 청크 수
    - RAG 실습 현황 (간단 / 고급 / 비교)
  </Tab>
  <Tab title="프롬프트 & 템플릿">
    **프롬프트 엔지니어링 학습 현황입니다.**

    - 학습 템플릿 활용 수
    - 직접 작성한 프롬프트 수
    - Few-shot 활용 현황
  </Tab>
  <Tab title="MCP 학습">
    **MCP 도구 활용 현황을 추적합니다.**

    - 생성한 MCP 수
    - 활성 도구 수
    - MCP 연동 대화 수
    - 에이전트 연결 수
    - 도구 카테고리별 사용 통계
  </Tab>
  <Tab title="에이전트 빌더">
    **에이전트 설계 및 활용 현황입니다.**

    - 설계한 에이전트 수
    - 사용된 총 노드 수
    - 노드 유형별 활용 분포
  </Tab>
</Tabs>

## 학습 관리 핵심 체크리스트

<AccordionGroup>
  <Accordion title="수강 강의 확인">
    현재 등록된 강의 목록과 수강 기간, 상태를 정기적으로 확인하세요. 수강 기간이 지난 강의는 학습 이력 접근이 제한될 수 있습니다.
  </Accordion>
  <Accordion title="학습 성과 모니터링">
    대시보드에서 각 기능의 활용 현황을 주기적으로 점검하세요. 사용이 적은 기능은 의도적으로 실습 기회를 늘려 균형 있는 학습을 유지할 수 있습니다.
  </Accordion>
  <Accordion title="대화 기록 검색 / 내보내기">
    중요한 AI 대화는 내보내기 기능을 활용하여 별도로 보관하세요. 키워드 검색으로 과거 실습 내용을 빠르게 찾아 복습할 수 있습니다.
  </Accordion>
  <Accordion title="학습 패턴 점검">
    어떤 모델을 가장 많이 쓰는지, 어떤 기능은 잘 사용하지 않는지를 돌아보세요. 학습 패턴을 파악하면 GrowFit 플랫폼을 더 효율적으로 활용할 수 있습니다.
  </Accordion>
</AccordionGroup>

## 수강 강의 등록 & 관리

<Steps>
  <Step title="설정 메뉴 접근">
    화면 우측 상단의 **프로필 아이콘** → **설정** → **수강 강의** 탭을 클릭합니다.
  </Step>
  <Step title="새 강의 등록">
    **[새 강의 등록]** 버튼을 클릭하고 강사로부터 받은 **6자리 초대코드**를 입력합니다.

    <Tip>
      초대코드는 **영문 대문자 6자리**로 구성됩니다. 강의 생성 후 강사가 직접 발급합니다. 대소문자를 정확히 구분하고, 앞뒤 공백이 없는지 확인하세요.
    </Tip>
  </Step>
  <Step title="강의 정보 확인">
    등록된 강의의 정보를 카드 형태로 확인합니다.

    | 항목 | 내용 |
    |---|---|
    | 강의명 | 등록된 강의의 이름 |
    | 강사명 | 담당 강사 이름 |
    | 수강 기간 | 강의 시작일 ~ 종료일 |
    | 상태 | 수강 중 / 수강 완료 / 만료 |
  </Step>
  <Step title="강의 삭제 (주의)">
    더 이상 필요 없는 강의는 카드의 **삭제 버튼**으로 제거할 수 있습니다.

    <Warning>
      강의를 삭제하면 해당 강의와 연결된 **학습 기록 및 데이터는 복구할 수 없습니다.** 삭제 전 반드시 필요한 내용을 내보내기하거나 메모해두세요.
    </Warning>
  </Step>
</Steps>

<Note>
  학습 성과 대시보드는 단순한 사용량 통계가 아닙니다. 내가 어떤 AI 기능을 얼마나 깊이 활용하고 있는지를 반영하는 **개인 AI 역량 지표**입니다. 대시보드를 꾸준히 확인하며 균형 있는 학습을 이어가세요.
</Note>
