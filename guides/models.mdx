## 모델 선정 철학

GrowFit이 19종의 다양한 AI 모델을 제공하는 데에는 명확한 이유가 있습니다.

> **"어떤 AI가 최고인가?"가 아니라 "이 작업에는 어떤 AI가 적합한가?"**

AI 모델마다 강점과 특성이 다릅니다. 코딩에 강한 모델, 긴 문서 분석에 강한 모델, 한국어에 최적화된 모델이 각각 다릅니다. GrowFit에서는 다양한 모델을 직접 비교 실험하며 **작업에 맞는 AI를 선택하는 안목**을 키울 수 있습니다.

<Note>
  GrowFit은 현재 **19종의 AI 모델**을 제공하며 지속적으로 확장하고 있습니다. 이 중 **무료 14종**은 비용 걱정 없이 반복 실험할 수 있습니다.
</Note>

## 범용 모델 vs 특화 모델

<Tabs>
  <Tab title="범용 모델">
    범용 모델은 **다양한 작업에 고르게 뛰어난 성능**을 발휘하는 모델입니다. 특정 작업에 치우치지 않고 글쓰기, 요약, 질문 답변, 코딩 등 전반적인 업무에 활용할 수 있습니다.

    **대표 모델**
    - GPT-4o (OpenAI)
    - Claude 3.7 Sonnet (Anthropic)
    - Gemini 1.5 Pro (Google)
    - Llama 3.1 70B (Meta)

    **추천 사용 상황**
    - 작업 유형이 다양하거나 명확하지 않을 때
    - 처음 AI를 사용해보는 경우
    - 일반적인 텍스트 작성 및 편집
  </Tab>
  <Tab title="특화 모델">
    특화 모델은 **특정 작업 영역에서 범용 모델을 능가하는 성능**을 발휘하도록 최적화된 모델입니다. 해당 작업에 한해 훨씬 뛰어난 결과를 얻을 수 있습니다.

    **대표 모델**
    - Qwen3 Coder — 코딩 특화
    - EXAONE 3.5 — 한국어 특화
    - Solar Pro 3 — 한국어 특화
    - DeepSeek R1 — 추론 특화
    - Gemini Flash Image — 이미지 생성 특화

    **추천 사용 상황**
    - 코드 작성 및 디버깅
    - 한국어 문서 작성 및 번역
    - 논리적 추론이 필요한 복잡한 문제
    - 이미지 분석 또는 생성
  </Tab>
</Tabs>

## 작업 유형별 추천 모델

아래 표를 참고하여 작업 목적에 맞는 AI 모델을 선택하세요.

| 작업 유형 | 추천 모델 | 추천 이유 |
|---|---|---|
| 코딩 / 개발 | Qwen3 Coder, Devstral 2, Claude | 코딩 특화 모델 |
| 창의적 글쓰기 | GPT, Gemini, Llama 70B | 풍부한 어휘력 |
| 긴 문서 분석 | Claude | 200K 토큰 문맥 처리 |
| 빠른 응답 | Gemini Flash, GPT-3.5, Nemotron Nano | 경량 모델 |
| 한국어 특화 | EXAONE, Solar Pro 3 | 한국어 강점 |
| 비용 효율 | 무료 14종 | 무료 반복 실험 |

## 특수 목적 모델

<CardGroup cols={3}>
  <Card title="추론 특화" icon="brain">
    **DeepSeek R1, Qwen3 Thinking**

    복잡한 수학 문제, 논리 퍼즐, 단계별 추론이 필요한 작업에 적합합니다. 답을 내기 전에 생각하는 과정(Chain-of-Thought)을 거칩니다.
  </Card>
  <Card title="이미지 분석" icon="image">
    **Qwen3 VL, Mistral Small**

    이미지를 입력하면 내용을 분석하고 설명합니다. 차트, 다이어그램, 사진의 내용을 텍스트로 변환하거나 설명하는 데 활용합니다.
  </Card>
  <Card title="이미지 생성" icon="wand-magic-sparkles">
    **Gemini Flash Image**

    텍스트 설명을 기반으로 이미지를 생성합니다. 프레젠테이션용 이미지, 아이디어 시각화, 콘텐츠 제작에 활용합니다.
  </Card>
</CardGroup>

## 모델 선택 가이드

<AccordionGroup>
  <Accordion title="어떤 모델을 먼저 써봐야 할까요?" icon="circle-question">
    처음이라면 **GPT-4o** 또는 **Claude 3.7 Sonnet**으로 시작하는 것을 권장합니다. 두 모델 모두 범용성이 뛰어나며, 대부분의 작업에서 안정적인 결과를 제공합니다.

    익숙해진 후에는 특정 작업에 특화된 모델을 시도해보세요.
  </Accordion>
  <Accordion title="한국어 작업에는 어떤 모델이 좋나요?" icon="language">
    **EXAONE 3.5**(LG AI Research)와 **Solar Pro 3**(Upstage)는 한국어 처리에 특화된 모델입니다. 한국어 문서 작성, 번역, 요약 작업에서 뛰어난 성능을 발휘합니다.

    글로벌 모델인 GPT, Claude, Gemini도 한국어를 지원하지만, 특화 모델에 비해 미묘한 뉘앙스나 존댓말 처리에서 차이가 날 수 있습니다.
  </Accordion>
  <Accordion title="무료 모델은 어떤 것들인가요?" icon="gift">
    GrowFit에서 제공하는 19종 중 **14종은 무료**로 사용할 수 있습니다. 비용 걱정 없이 반복적으로 실험하며 모델 특성을 체험해보세요.

    무료 모델에는 Llama 시리즈, EXAONE, Solar, DeepSeek, Qwen 시리즈 등이 포함되어 있습니다.
  </Accordion>
  <Accordion title="코딩 작업에는 어떤 모델이 좋나요?" icon="code">
    **Qwen3 Coder**와 **Devstral 2**는 코드 생성, 디버깅, 코드 리뷰에 특화된 모델입니다. 일반 범용 모델에 비해 더 정확한 코드를 생성하고 오류를 잘 찾아냅니다.

    **Claude** 역시 코딩 능력이 뛰어나며, 코드 설명과 문서화에도 강점을 보입니다.
  </Accordion>
  <Accordion title="빠른 응답이 필요한 경우에는?" icon="bolt">
    **Gemini Flash**, **GPT-3.5**, **Nemotron Nano** 등의 경량 모델은 응답 속도가 빠르고 비용 효율적입니다. 간단한 질문 답변, 초안 작성, 빠른 아이디어 탐색에 활용하세요.
  </Accordion>
</AccordionGroup>

## 모델 비교 실험 방법

GrowFit에서 모델을 직접 비교해보려면 **LLM 실습** 메뉴를 활용하세요.

<Steps>
  <Step title="LLM 실습 메뉴 이동">
    사이드바에서 **LLM 실습(Step 2)** 을 클릭합니다.
  </Step>
  <Step title="모델 선택">
    상단의 모델 선택 드롭다운에서 비교할 AI 모델을 선택합니다.
  </Step>
  <Step title="동일한 질문 입력">
    같은 질문이나 작업을 각 모델에 입력하고 결과를 비교합니다.
  </Step>
  <Step title="파라미터 조정">
    Temperature, Top-P 등의 파라미터를 조정하며 응답 변화를 관찰합니다.
  </Step>
  <Step title="결과 분석">
    각 모델의 답변 품질, 속도, 스타일 차이를 비교하고 작업에 적합한 모델을 선정합니다.
  </Step>
</Steps>

<Tip>
  같은 프롬프트로 3~5개 모델을 비교해보면 모델 간 차이를 명확하게 체감할 수 있습니다. GrowFit의 무료 모델을 활용하면 비용 부담 없이 충분히 실험할 수 있습니다.
</Tip>
