## 자주 묻는 질문

<AccordionGroup>
  <Accordion title="어떤 모델을 선택해야 할까요?">
    **작업의 목적과 복잡도에 따라 선택하는 것이 가장 좋습니다.**

    | 상황 | 추천 모델 |
    |---|---|
    | 빠른 응답이 필요할 때 | GPT-4o mini, Claude 3.5 Haiku |
    | 복잡한 분석이나 심층 추론 | GPT-4o |
    | 모델 선택이 확신되지 않을 때 | 비교 모드로 2~3개 동시 비교 |

    <Tip>
      어떤 모델이 더 잘 맞는지 확신이 없다면, 비교 모드에서 2~3개 모델에 같은 질문을 던져보세요. 실제 답변을 비교하면 금방 판단할 수 있습니다.
    </Tip>
  </Accordion>

  <Accordion title="한 번에 몇 개의 모델까지 비교할 수 있나요?">
    **최대 3개의 모델을 동시에 비교할 수 있습니다.**

    - **2개 이상** 선택 시 자동으로 비교 모드로 전환됩니다.
    - **1개** 선택 시 단일 모드로 대화합니다.
    - **3개** 선택 시 3개 패널이 나란히 표시됩니다.
  </Accordion>

  <Accordion title="초대코드를 받지 못했어요.">
    다음 순서로 확인해보세요.

    1. **강사에게 직접 문의**하여 초대코드 재발급을 요청하세요.
    2. **이메일 스팸함**을 확인하세요. 자동 분류로 스팸함에 들어갔을 수 있습니다.
    3. 초대코드 입력 시 **대소문자를 정확히 구분**하고, **앞뒤 공백이 없는지** 확인하세요.

    <Note>
      초대코드는 영문 대문자 6자리입니다. 예: `ABCD12` 형식
    </Note>
  </Accordion>

  <Accordion title="이미지 분석·생성 기능은 언제 사용할 수 있나요?">
    **멀티모달 기능은 현재 출시 준비 중입니다.**

    준비 중인 기능은 다음과 같습니다.

    | 기능 | 설명 |
    |---|---|
    | 비전 분석 | 이미지를 업로드하면 내용을 분석하여 텍스트/태그로 출력 |
    | 이미지 생성 | 텍스트 프롬프트를 입력하면 이미지 URL 생성 |
    | OCR 추출 | 이미지 속 텍스트를 디지털 텍스트로 변환 |

    출시 전까지는 **텍스트 기반 기능(LLM 실습, RAG, MCP, 에이전트 빌더)을 먼저 익혀두는 것**을 권장합니다. 멀티모달 기능이 출시되면 훨씬 더 풍부하게 활용할 수 있습니다.
  </Accordion>

  <Accordion title="만든 에이전트를 학생들과 공유할 수 있나요?">
    **가능합니다.** 다음 순서로 공유하세요.

    <Steps>
      <Step title="게시 상태 변경">
        에이전트를 **Published(게시됨)** 상태로 변경합니다.
      </Step>
      <Step title="공유 옵션 활성화">
        에이전트 설정에서 **공유 옵션을 켜고** 링크를 복사합니다.
      </Step>
      <Step title="링크 배포">
        복사한 링크를 수강생에게 배포합니다. 링크를 받은 수강생은 에이전트를 바로 사용할 수 있습니다.
      </Step>
    </Steps>

    <Note>
      공유된 에이전트의 **사용 통계와 대화 이력은 강사가 모니터링**할 수 있습니다. 학습 현황 파악에 활용하세요.
    </Note>
  </Accordion>
</AccordionGroup>
