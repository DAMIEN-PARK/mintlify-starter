## MCP란 무엇인가?

**MCP(Model Context Protocol)**는 AI 모델이 외부 도구를 호출하는 표준 방식입니다. 한마디로 **AI에게 도구 상자를 쥐여주는 기능**입니다.

AI는 학습된 지식에만 의존하는 것이 아니라, MCP를 통해 **실시간 데이터 조회와 외부 도구 사용**이 가능해져 훨씬 더 정확한 답변을 제공합니다.

<Note>
  GrowFit 플랫폼에서 MCP는 **별도 설정 없이 바로 사용**할 수 있습니다.
</Note>

---

## MCP 작동 프로세스

```
사용자 질문 → AI 모델(LLM) → MCP 표준 프로토콜 → 외부 도구(날씨 API, 검색엔진, 계산기 등) → 답변 생성
```

MCP 표준 프로토콜을 통해 AI와 외부 시스템이 일관된 방식으로 통신합니다.

## MCP vs 에이전트 비교

<CardGroup cols={2}>
  <Card title="MCP (도구)" icon="wrench">
    **AI의 손과 발**

    - 외부 시스템 연결
    - 실시간 데이터 확인
    - 계산 / 파일 처리

    *칼과 불 같은 도구*
  </Card>
  <Card title="에이전트 (두뇌)" icon="brain">
    **AI의 판단과 계획**

    - 시스템 프롬프트로 역할 설정
    - 작업 계획 및 추론
    - 의도 파악 및 대화 관리

    *요리사처럼 방향을 결정*
  </Card>
</CardGroup>

<Tip>
  최강 조합은 **요리사(에이전트) + 칼과 불(MCP)**입니다. 에이전트가 방향을 결정하고, MCP가 실행을 담당합니다.
</Tip>

## MCP ON/OFF 차이 비교

MCP를 켜면 AI의 답변이 얼마나 달라지는지 확인해보세요.

| 질문 유형 | MCP OFF | MCP ON |
|---|---|---|
| 현재 시간 | "실시간 정보에 접근할 수 없습니다" | "서울 기준 2026년 2월 26일 오후 4시 30분입니다" |
| 날씨 | "인터넷에 접속할 수 없습니다" | "서울 현재 15°C, 맑음" |
| 수학 계산 | "약 9천 7백만 정도" (부정확) | "정확히 97,406,784" |
| 환율 | "2023년까지의 지식만 있습니다" | "현재 원/달러 1,450.50원" |

## 16가지 플랫폼 도구

GrowFit 플랫폼에서 제공하는 MCP 도구를 카테고리별로 정리했습니다.

<Tabs>
  <Tab title="금융 / 데이터">
    | 도구 | 설명 |
    |---|---|
    | DART 전자공시 (재무) | 기업 재무제표 및 재무 데이터 조회 |
    | DART 전자공시 (공시) | 기업 공시 정보 및 경영 공시 조회 |
  </Tab>
  <Tab title="웹 / 정보">
    | 도구 | 설명 |
    |---|---|
    | 웹 검색 | 실시간 인터넷 검색 |
    | 위키피디아 검색 | 위키피디아 문서 검색 |
    | 웹 크롤링 | 특정 URL의 웹 페이지 내용 수집 |
  </Tab>
  <Tab title="유틸리티">
    | 도구 | 설명 |
    |---|---|
    | 환율 조회 | 실시간 환율 정보 |
    | 날씨 조회 | 현재 날씨 및 예보 |
    | 계산기 | 수식 계산 (정확한 수치) |
    | QR 코드 생성 | URL 또는 텍스트를 QR 코드로 변환 |
    | 날짜 / 시간 조회 | 현재 날짜 및 시간 조회 |
  </Tab>
  <Tab title="파일 / 언어 / 시각화">
    | 카테고리 | 도구 | 설명 |
    |---|---|---|
    | 파일 처리 | 파일 읽기 | 업로드된 파일 내용 읽기 |
    | 언어 / 학습 | 맞춤법 검사 | 한국어 맞춤법 교정 |
    | 언어 / 학습 | 단어장 | 단어 저장 및 관리 |
    | 언어 / 학습 | 퀴즈 생성 | 학습 퀴즈 자동 생성 |
    | 언어 / 학습 | 문법 교정 | 문법 오류 수정 |
    | 시각화 | 차트 시각화 | 데이터를 차트로 시각화 |
  </Tab>
</Tabs>

## 7가지 MCP 템플릿

목적에 맞는 도구 조합을 바로 사용할 수 있는 사전 구성 템플릿입니다.

<AccordionGroup>
  <Accordion title="1. 학습 도우미">
    **포함 도구**: 웹 검색, 계산기, 날짜/시간 조회

    학습 중 모르는 내용을 즉시 검색하고, 수식 계산과 날짜 계산까지 한 번에 처리할 수 있습니다.
  </Accordion>
  <Accordion title="2. 리서치">
    **포함 도구**: 웹 검색, 웹 크롤링, 파일 읽기

    최신 정보 수집부터 특정 페이지 분석, 업로드 파일 내용 참조까지 조사 업무에 최적화된 구성입니다.
  </Accordion>
  <Accordion title="3. 코딩 연습">
    **포함 도구**: 계산기, 웹 검색, 날짜/시간 조회

    코드 작성 중 필요한 수치 계산, 레퍼런스 검색, 날짜 처리를 지원합니다.
  </Accordion>
  <Accordion title="4. 글쓰기">
    **포함 도구**: 웹 검색, 파일 읽기

    글쓰기에 필요한 자료 검색과 참고 문서 내용을 AI가 바로 활용할 수 있습니다.
  </Accordion>
  <Accordion title="5. 수학 / 과학">
    **포함 도구**: 계산기, 웹 검색

    정확한 수식 계산과 관련 개념 검색으로 수학·과학 문제 해결을 도와줍니다.
  </Accordion>
  <Accordion title="6. 외국어 학습">
    **포함 도구**: 웹 검색, 웹 크롤링

    외국어 표현 검색과 실제 원문 사이트 내용을 수집하여 자연스러운 언어 학습을 지원합니다.
  </Accordion>
  <Accordion title="7. 자유 구성">
    **포함 도구**: 모든 도구 선택 가능

    위의 16가지 도구 중 원하는 것을 자유롭게 조합하여 나만의 MCP를 구성합니다.
  </Accordion>
</AccordionGroup>

<Note>
