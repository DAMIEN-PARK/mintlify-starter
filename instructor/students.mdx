## 학생 초대하기

<Steps>
  <Step title="학생 초대 버튼 클릭">
    좌측 사이드바에서 **[학생 관리]** 를 클릭합니다. 학생 목록 페이지 우측 상단의 **[학생 초대]** 버튼을 클릭합니다.
  </Step>
  <Step title="강의 선택">
    초대 팝업에서 학생을 초대할 **강의를 선택**합니다. 선택한 강의에 해당하는 초대 코드가 팝업 화면에 표시됩니다.

    <Note>
      개설된 강의가 여러 개인 경우, 학생이 참여할 정확한 강의를 선택하세요. 강의별로 초대 코드가 다릅니다.
    </Note>
  </Step>
  <Step title="초대 코드 공유">
    표시된 초대 코드 또는 초대 링크를 학생들에게 공유합니다.

    - **[코드 복사]** 버튼으로 초대 코드만 복사
    - **[링크 복사]** 버튼으로 직접 참여 가능한 초대 링크 복사
    - 이메일, 메신저, LMS, 카카오톡 등 다양한 방법으로 공유 가능

    <Tip>
      초대 링크를 공유하면 학생들이 코드를 직접 입력할 필요 없이 링크 클릭만으로 강의에 바로 참여할 수 있어 더 편리합니다.
    </Tip>
  </Step>
</Steps>

## 학생 목록 테이블

학생 목록에서 강의에 참여한 전체 학생을 조회합니다.

| 컬럼 | 설명 |
|------|------|
| 학생명 | 학생이 등록한 표시 이름 |
| 이메일 | 가입 시 사용한 이메일 주소 |
| 수강 강의 | 참여 중인 강의명 |
| 최근 접속일 | 마지막 로그인 날짜 |
| 총 대화 수 | 누적 AI 대화 건수 |
| 상태 | 활성 / 비활성 배지 |
| 상세보기 | 개별 학생 상세 리포트로 이동 |

<Tabs>
  <Tab title="강의별 필터">
    강의 목록 상단의 강의 드롭다운에서 특정 강의를 선택하면 해당 강의 수강생만 필터링하여 조회할 수 있습니다.
  </Tab>
  <Tab title="학생 검색">
    검색창에 학생 이름 또는 이메일 일부를 입력하면 해당 학생을 빠르게 찾을 수 있습니다.
  </Tab>
  <Tab title="정렬 옵션">
    컬럼 헤더를 클릭하면 해당 기준으로 오름차순/내림차순 정렬이 가능합니다. 최근 접속일, 총 대화 수 기준 정렬을 활용해 학습 참여도를 파악하세요.
  </Tab>
</Tabs>

## 학생 관리 활용 팁

  <Accordion title="저참여 학생 파악하기" icon="triangle-exclamation">
    최근 접속일 기준으로 정렬하면 장기간 접속하지 않은 학생을 쉽게 파악할 수 있습니다. 해당 학생들에게 별도로 연락하여 학습을 독려하세요.

  </Accordion>
</AccordionGroup>
