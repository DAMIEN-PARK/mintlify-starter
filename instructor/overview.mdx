## 강사 모드란?

GrowFit 강사 모드는 교육자를 위한 전용 운영 환경입니다. 강의 개설부터 학생 관리, 학습 자료 배포, AI 에이전트 설계까지 수업 운영에 필요한 모든 기능을 통합 제공합니다.

---

## 강사 모드 주요 역할 6가지

<CardGroup cols={2}>
  <Card title="① 강의 개설 및 운영" icon="chalkboard">
    강의 기간 설정, 학생 수 제한, AI 모델 선택 등 강의 환경을 구성합니다. 강의별로 독립적인 학습 공간을 제공합니다.
  </Card>
  <Card title="② 학생 초대 및 관리" icon="users">
    초대 코드 및 링크를 발급하여 학생을 강의에 초대합니다. 학생별 학습 진도와 활동 내역을 실시간으로 조회합니다.
  </Card>
  <Card title="③ 학습 자료 제공" icon="book-open">
    에이전트, 프롬프트 템플릿, Few-shot 예시 세트를 작성하고 학생들에게 배포합니다.
  </Card>
  <Card title="④ MCP 구성 및 배포" icon="plug">
    외부 도구 조합(MCP)을 생성하고 학생 환경에 배포합니다. 실습에 필요한 도구 환경을 미리 설정해 줄 수 있습니다.
  </Card>
  <Card title="⑤ 에이전트 설계" icon="diagram-project">
    노드 기반 워크플로우 빌더를 사용해 커스텀 AI 에이전트를 설계하고 학생들에게 배포합니다.
  </Card>
  <Card title="⑥ 사용량 모니터링" icon="chart-bar">
    AI 모델별 토큰 사용량을 분석하고 학생별 리포트를 시각화합니다. 수업 참여도와 비용을 통합 관리합니다.
  </Card>
</CardGroup>

## 화면 구성 한눈에 보기

강사 모드 화면은 크게 세 영역으로 구성됩니다.

<CardGroup cols={3}>
  <Card title="① 사이드바 (좌측)" icon="sidebar">
    **강의 운영, 교육 콘텐츠 관리, 시스템 설정** 으로 이동하는 주요 내비게이션 영역입니다.

    - 강의 등록 / 학생 관리
    - 사용량 통계
    - 프롬프트 템플릿 / Few-shot
    - MCP 빌더 / 에이전트 빌더
  </Card>
  <Card title="② 메인 영역 (중앙)" icon="window-maximize">
    선택한 메뉴의 **상세 기능이 표시되는 영역**입니다.

    - 강의 목록 및 설정 화면
    - 학생 현황 테이블
    - 통계 차트 및 리포트
    - 빌더 편집 캔버스
  </Card>
  <Card title="③ 상단 메뉴" icon="bars">
    각 메뉴에 해당하는 **빠른 실행 버튼 및 카테고리**를 확인할 수 있는 영역입니다.

    - 현재 위치 표시 (브레드크럼)
    - 주요 액션 버튼
    - 프로필 및 설정 접근
  </Card>
</CardGroup>

## 강사 사이드바 메뉴 상세

<AccordionGroup>
  <Accordion title="강의 등록" icon="chalkboard">
    개설한 강의 목록을 조회하고 새 강의를 등록합니다. 강의별 초대 코드를 확인하고 강의 상태(진행 중/종료)를 관리합니다.
  </Accordion>
  <Accordion title="학생 관리" icon="users">
    강의에 등록된 학생 목록을 확인합니다. 학생별 접속 현황, 학습 활동 지표, 상세 리포트를 조회할 수 있습니다.
  </Accordion>
  <Accordion title="사용량 통계" icon="chart-line">
    일간/주간/월간 단위로 AI 사용량을 분석합니다. 모델별 사용 비중, 강의별 통계, 학생별 순위를 확인합니다.
  </Accordion>
  <Accordion title="프롬프트 템플릿 / Few-shot" icon="pen-nib">
    학생들이 실습에 활용할 수 있는 프롬프트 템플릿과 Few-shot 예시 세트를 작성하고 강의별로 배포합니다.
  </Accordion>
  <Accordion title="MCP 빌더" icon="plug">
    플랫폼이 제공하는 16가지 MCP 도구를 조합하여 학습 목적에 맞는 도구 세트를 만들고 학생에게 공유합니다.
  </Accordion>
  <Accordion title="에이전트 빌더" icon="diagram-project">
    노드 기반 시각적 편집기로 AI 워크플로우를 설계합니다. 설계한 에이전트를 저장하고 강의에 배포합니다.
  </Accordion>
</AccordionGroup>

## 강사 모드 전환 및 학생 화면 미리보기

<Tabs>
  <Tab title="강사 모드 접속">
    승인된 강사 계정으로 로그인하면 자동으로 강사 모드 대시보드가 표시됩니다.

    - 로그인 후 강사 대시보드 자동 진입
    - 상단 또는 프로필 영역에서 현재 모드 확인 가능
    - 강사 전용 사이드바 메뉴가 활성화됨
  </Tab>
  <Tab title="학생 화면 미리보기">
    학생이 실제로 보는 화면을 강사 계정에서 미리 확인할 수 있습니다.

    <Tip>
      프로필 버튼 메뉴 안의 **[학생 화면 보기]** 를 클릭하면 학생 관점의 인터페이스를 미리 확인할 수 있습니다. 수업 전 학생들이 겪을 수 있는 UX를 직접 체험해 보세요.
    </Tip>

    - 학생 화면 미리보기 모드에서는 실제 데이터를 수정할 수 없습니다.
    - 화면 상단의 **[강사 모드로 돌아가기]** 버튼으로 언제든 복귀합니다.
  </Tab>
</Tabs>

## 강사 시작 체크리스트

<Steps>
  <Step title="강사 계정 승인 확인">
    이메일로 승인 완료 알림을 받았는지 확인합니다. 승인 완료 후 강사 대시보드에 접근 가능합니다.
  </Step>
  <Step title="첫 강의 개설">
    사이드바 **[강의 등록]** 메뉴에서 첫 강의를 개설하고 초대 코드를 확인합니다.
  </Step>
  <Step title="학생 초대">
    발급된 초대 코드를 학생들에게 공유하여 강의에 참여시킵니다.
  </Step>
  <Step title="학습 자료 준비">
    **[프롬프트 템플릿]** 또는 **[MCP 빌더]** 에서 수업에 사용할 자료를 미리 준비합니다.
  </Step>
  <Step title="학생 화면 미리보기">
    프로필 메뉴의 **[학생 화면 보기]** 로 학생 관점에서 인터페이스를 최종 점검합니다.
  </Step>
</Steps>
