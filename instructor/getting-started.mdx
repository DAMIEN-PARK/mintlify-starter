## 역할 구분

GrowFit에는 두 가지 역할이 있으며, 역할에 따라 사용 가능한 기능과 가입 절차가 다릅니다.

<CardGroup cols={2}>
  <Card title="훈련생 (Trainee)" icon="user-graduate">
    **가입 즉시 사용 가능**

    - 별도의 승인 절차 없이 바로 사용 가능
    - 강사로부터 받은 초대 코드로 강의 참여
    - LLM 실습, 프롬프트, RAG, MCP, 에이전트 빌더 등 모든 학습 기능 이용
    - 학습 기록 및 성과 대시보드 확인
  </Card>
  <Card title="강사 (Instructor)" icon="chalkboard-user">
    **관리자 승인 필수**

    - 강의 개설 및 학생 관리 권한 보유
    - 수강생 초대 코드 발급 및 진도 모니터링
    - 학습 자료 및 에이전트 배포
    - **가입 후 관리자의 승인이 완료되어야 강사 기능 활성화**
  </Card>
</CardGroup>

<Warning>
  강사 계정은 관리자 승인 전까지 강사 전용 기능(강의 관리, 수강생 모니터링, 콘텐츠 배포 등)을 사용할 수 없습니다. 승인 완료 전에는 수강생과 동일한 기능만 이용 가능합니다.
</Warning>

## 강사 신청 필수 정보

강사 신청 시 아래 정보를 미리 준비하세요.

<CardGroup cols={2}>
  <Card title="이름" icon="id-card">
    실명을 입력합니다. 학생 관리 화면에 표시되는 이름입니다.
  </Card>
  <Card title="이메일 (인증 필수)" icon="envelope">
    인증 메일을 수신할 수 있는 유효한 이메일 주소를 사용하세요.
  </Card>
  <Card title="전화번호" icon="phone">
    연락 가능한 휴대폰 번호를 입력합니다.
  </Card>
  <Card title="소속" icon="building">
    소속 기관 또는 단체명을 입력합니다. 소속이 없는 경우 **'무소속 과정'** 을 입력하세요.
  </Card>
</CardGroup>

## 회원가입 및 승인 절차

<Steps>
  <Step title="웹사이트 접속">
    브라우저를 열고 **[growfit.kr](https://growfit.kr)** 에 접속합니다. 메인 화면 상단 우측의 **[회원가입]** 버튼을 클릭하세요.
  </Step>
  <Step title="이메일 인증">
    회원가입 페이지에서 필수 정보를 입력합니다. **[인증 메일 발송]** 버튼을 클릭하면 입력한 이메일로 인증 메일이 발송됩니다. 받은 편지함을 확인하고 메일 내 **인증 링크를 클릭**하여 이메일 인증을 완료하세요.

    <Warning>
      인증 링크는 발송 후 **10분 이내에** 클릭해야 합니다. 시간이 초과되면 재발송 버튼을 눌러 새 인증 메일을 받으세요.
    </Warning>
  </Step>
  <Step title="역할 선택">
    이메일 인증 완료 후 최초 로그인 시 역할 선택 화면이 나타납니다. **[강사(Instructor)]** 를 선택하세요.

    <Note>
      역할은 최초 선택 후 변경이 어려울 수 있으므로 신중하게 선택하세요.
    </Note>
  </Step>
  <Step title="강사 신청 정보 입력">
    강사 신청 양식에서 아래 정보를 정확하게 입력합니다.

    - **이름**: 실명 입력
    - **이메일**: 가입 시 사용한 이메일 (자동 입력)
    - **전화번호**: 연락 가능한 휴대폰 번호
    - **소속**: 소속 기관명 (없는 경우 '무소속 과정')

    모든 정보 입력 후 **[신청 완료]** 버튼을 클릭하세요.
  </Step>
</Steps>

## 승인 안내

강사 신청 후 검토 및 승인 절차가 진행됩니다.

<CardGroup cols={3}>
  <Card title="소요 시간" icon="clock">
    영업일 기준 **1~2일** 이내 처리됩니다.
  </Card>
  <Card title="승인 알림" icon="bell">
    승인 완료 시 가입한 **이메일로 알림**이 발송됩니다.
  </Card>
  <Card title="기능 활성화" icon="unlock">
    승인 이후 **강사 대시보드**에 접근하고 모든 강사 기능을 사용할 수 있습니다.
  </Card>
</CardGroup>

<Tip>
  승인 대기 중에도 수강생 모드로 GrowFit의 학습 기능을 자유롭게 체험해 볼 수 있습니다. 먼저 플랫폼에 익숙해지는 시간으로 활용하세요.
</Tip>

## 계정 보안 및 로그아웃

<AccordionGroup>
  <Accordion title="비밀번호 관리" icon="lock">
    보안이 강화된 비밀번호를 사용하여 강사 계정을 안전하게 보호하세요.

    - 비밀번호는 **8자 이상**, 영문·숫자·특수문자를 조합하여 설정하세요.
    - 비밀번호 변경은 **프로필 > 계정 설정 > 비밀번호 변경** 메뉴에서 가능합니다.
    - 현재 비밀번호를 입력한 후 새 비밀번호를 설정합니다.
    - 주기적으로 비밀번호를 변경하는 것을 권장합니다.
  </Accordion>
  <Accordion title="비밀번호 분실 시" icon="key">
    로그인 화면 하단의 **[비밀번호 찾기]** 를 클릭하고 가입한 이메일을 입력하면 재설정 링크가 발송됩니다. 메일 내 링크를 클릭하여 새 비밀번호를 설정하세요.
  </Accordion>
  <Accordion title="안전한 로그아웃 절차" icon="right-from-bracket">
    강사 계정에는 학생 데이터가 포함되어 있으므로 반드시 안전하게 로그아웃하세요.

    1. 화면 좌측 하단의 **프로필 아이콘** 클릭
    2. 팝업 메뉴에서 **[로그아웃]** 클릭
    3. 로그인 화면으로 이동하면 로그아웃 완료

    <Warning>
      공용 컴퓨터나 공개 네트워크에서 사용한 경우 반드시 로그아웃하세요. 강사 계정에는 학생 개인정보가 포함되어 있어 보안이 중요합니다.
    </Warning>
  </Accordion>
  <Accordion title="2단계 인증 (권장)" icon="shield-halved">
    계정 보안을 강화하려면 **프로필 > 보안 설정** 에서 2단계 인증을 활성화하세요. 로그인 시 이메일 또는 인증 앱을 통한 추가 인증이 요구됩니다.
  </Accordion>
</AccordionGroup>
