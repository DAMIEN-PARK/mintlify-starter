## 사용량 통계 페이지 개요

사이드바의 **[사용량 통계]** 메뉴에서 강의 전반의 AI 사용 현황을 종합적으로 분석할 수 있습니다. 기간별, 강의별, 모델별, 학생별로 세분화된 데이터를 제공합니다.

---

## 페이지 구성

### 1. 기간 필터 탭

페이지 상단의 탭으로 조회 기간을 전환합니다.

<Tabs>
  <Tab title="일간">
    오늘 하루의 AI 대화 현황을 시간대별로 확인합니다. 수업 진행 당일 실시간 참여 현황을 모니터링할 때 유용합니다.
  </Tab>
  <Tab title="주간">
    최근 7일간의 사용 추이를 일별로 확인합니다. 요일별 학습 패턴을 파악하는 데 활용하세요.
  </Tab>
  <Tab title="월간">
    이번 달의 사용 현황을 주별로 확인합니다. 강의 진행에 따른 학습 참여도 변화를 파악할 수 있습니다.
  </Tab>
  <Tab title="전체">
    강의 개설 이후 전체 기간의 누적 데이터를 조회합니다. 강의 종료 후 최종 성과 분석에 활용하세요.
  </Tab>
</Tabs>

---

### 2. 강의 필터

페이지 우측 상단의 드롭다운에서 특정 강의를 선택하여 해당 강의의 통계만 필터링할 수 있습니다.

- **전체 강의**: 모든 강의의 합산 통계 표시
- **특정 강의 선택**: 해당 강의 수강생의 통계만 표시

### 4. 일별 막대 차트

날짜별 AI 대화 건수 추이를 시각적으로 확인합니다.

- 각 막대는 해당 날짜의 총 대화 건수를 나타냅니다.
- 막대 위에 마우스를 올리면 상세 수치를 확인할 수 있습니다.
- **드래그로 확대**: 특정 기간을 드래그하면 해당 구간을 확대하여 세부 데이터를 조회할 수 있습니다.

<Tip>
  수업 진행 날짜에 막대가 높게 표시되는지 확인하세요. 수업일과 비수업일의 활동 차이를 통해 자기 주도 학습 참여율을 가늠할 수 있습니다.
</Tip>

### 5. AI 모델별 사용 현황

학생들이 어떤 AI 모델을 얼마나 사용했는지 파악합니다.

| AI 모델 | 사용 횟수 | 비중 |
|---------|-----------|------|
| GPT-4 | 예: 450건 | 예: 45% |
| Claude 3.5 | 예: 300건 | 예: 30% |
| GPT-3.5 Turbo | 예: 150건 | 예: 15% |
| 기타 모델 | 예: 100건 | 예: 10% |

<Note>
  실제 표시 데이터는 강의 설정에서 허용한 AI 모델과 학생들의 실제 사용에 따라 달라집니다.
</Note>

## 강의별 사용량 테이블

여러 강의를 운영 중인 경우, 강의별 사용량을 비교하는 테이블을 제공합니다.

| 컬럼 | 설명 |
|------|------|
| 강의명 | 개설된 강의 이름 |
| 학생 수 | 해당 강의 참여 학생 수 |
| 총 대화 수 | 강의 전체 AI 대화 건수 |
| 학생당 평균 | 수강생 1인당 평균 대화 수 |
| 상태 | 진행 중 / 종료 배지 |

<Tip>
  **강의명을 클릭**하면 해당 강의 학생들의 개별 사용량이 아래에 펼쳐집니다. 어떤 학생이 가장 활발하게 참여하고 있는지 바로 확인할 수 있습니다.
</Tip>

## 학생별 사용량

### 참여도 기반 순위

학생들의 총 대화 건수를 기준으로 자동 정렬되어 참여도 순위를 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| 순위 | 메달(1~3위) 또는 숫자로 표시 |
| 학생명 | 학생 표시 이름 |
| 총 대화 수 | 누적 AI 대화 건수 (정렬 기준) |
| 최근 활동 | 마지막으로 AI 대화를 한 날짜 |
| 상세보기 | 개별 학생 학습 이력 페이지로 이동 |

<AccordionGroup>
  <Accordion title="순위 표시 방식" icon="medal">
    - **1위**: 금메달 아이콘
    - **2위**: 은메달 아이콘
    - **3위**: 동메달 아이콘
    - **4위 이하**: 숫자 순위 표시
  </Accordion>
  <Accordion title="상세보기 활용" icon="magnifying-glass">
    우측의 **[상세보기]** 버튼을 클릭하면 해당 학생의 개별 학습 이력 페이지로 이동합니다. 수강 정보, 활동 지표, AI 실습 현황, 학습 활동 요약을 모두 확인할 수 있습니다.

    자세한 내용은 **[학생 관리](/instructor/students)** 페이지의 '학생 상세 리포트' 섹션을 참고하세요.
  </Accordion>
  <Accordion title="참여도 관리 활용" icon="chart-line">
    순위가 하위권이거나 최근 활동이 오래된 학생을 파악하여 적극적으로 학습을 독려하세요. 단, 참여도만으로 학습 성과 전체를 평가하지 않도록 주의하세요.
  </Accordion>
</AccordionGroup>

## 데이터 내보내기

<Tabs>
  <Tab title="CSV 내보내기">
    통계 페이지 우측 상단의 **[내보내기]** 버튼을 클릭하면 현재 조회 중인 데이터를 CSV 파일로 다운로드할 수 있습니다.

    - 강의별 요약 데이터
    - 학생별 상세 사용량 데이터
    - 기간 필터가 적용된 상태로 내보내기

    <Note>
      내보낸 데이터는 개인정보가 포함되어 있으므로 안전하게 관리하세요.
    </Note>
  </Tab>
  <Tab title="리포트 공유">
    특정 학생의 상세 리포트 페이지에서 **[리포트 공유]** 기능을 사용하면 해당 학생의 학습 성과 요약을 공유할 수 있습니다.
  </Tab>
</Tabs>
